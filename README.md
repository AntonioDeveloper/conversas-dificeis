## Conversas Difíceis (Difficult Conversations)
An AI tool that helps you understand how your message might be perceived before sending it.

![Print of the project](https://github.com/AntonioDeveloper/conversas-dificeis/blob/main/public/assets/images/printscreen.jpg)

## 🚀 Live Demo

[Live Demo](https://conversas-dificeis.vercel.app/)

## Technology 

Here are the technologies used in this project.

* Next.js
* React.js
* Typescript 
* Groq (LLM Model: openai/gpt-oss-120b)
* Tailwind
* ShadCN
* Node.js
* Express.js
* Socket.IO

## Services Used

* Github
* Vercel
* Render

## Getting started

- Clone the repository  

* Install dependencies:
>    npm install

* Run backend (server):
>    cd server
>    npm run dev

* Run frontend:
>    In another terminal, run npm run dev

* Project view:
>    Open a browser of your choice and go to localhost:3000
